# 🚗 Car Quiz Game

This is a simple browser-based JavaScript quiz game about cars.

I made this project with my brother Chaitany, who is currently in 7th standard.  
The idea was to spend some quality time learning together and help him get his hands off the tablet to work on something creative and interesting.

## What this project does
- Asks fun, Gen-Z style car-related questions
- Takes user input through prompts
- Shows the final score at the end

## Tech Used
- HTML
- CSS
- JavaScript (basic browser JavaScript)

## How to Run
1. Open `index.html` in any web browser
2. Click **Start Quiz**
3. Answer the questions and see your score

This project is beginner-friendly and meant for learning by doing.
